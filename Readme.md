### npm-update-all
a command line tool to update all dependences & devDependences in package.json

### install
```bash
npm install npm-update-all -g
```

### usage
```bash
npm-update-all
```

### usage with params
```bash
npm-update-all --registry=http://registry.npm.taobao.org
```
